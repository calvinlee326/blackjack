# Blackjack

A browser-based Blackjack game with a casino-style UI.

**Live Demo:** [bbblackjack.netlify.app](https://bbblackjack.netlify.app/)

## How to Play

1. Click **DEAL** to start a new round
2. Click **HIT** to draw another card
3. Try to reach 21 without going over — Blackjack wins!
